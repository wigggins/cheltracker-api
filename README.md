## Running Locally

Create a .env file in the root of your project, and add:

```js
JWT_SECRET=<supersupersecret>
JWT_EXPIRE=30d
MONGOURI=<add_you_mongodb_connection_uri>
```

Run <code>npm i</code> for initial installation, and <code>npm run dev</code> to run server.
